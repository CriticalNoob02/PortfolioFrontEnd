Bem Vindo ao meu primeiro Projeto Front-End!!

Projeto criado para o final do curso de "Web Development - FrontEnd", foi pedido para criar alguma pagina de tema livre,então decidi criar uma lojinha de calçados, logo apos fui atrás de umas cores diferentes para produzir a página, ao concluir a paleta de cores iniciei a modelagem do site, primeiramente criando uma barra de navegação que fosse responsiva e depois um banner estilo carrossel.
Com o início do site pronto chegou a hora de iniciar as demais abas do site, decidi colocar a aba "Sobre" logo no início para mostrar um pouco de credibilidade para a loja e depois chegou a parte mais interessante de trabalhar, os cards de produtos, para criar eles levei como inspiração o modo que os cards da loja virtual da Adidas se comportam, onde o efeito de hover leva a outra foto do mesmo produto porem decidi ocultar a informação do nome também, para deixar o card inicial mais limpo e com o foco apenas no produto. Para finalizar chegou a hora da aba de contatos onde criei um formulário básico e padrão.
Claramente existem alguns bugs ainda nessa página, como o hover do botão do formulário, que dependendo de como o mouse passa ele pisca varias vezes antes de encher, e a aba hamburguer da barra de navegação, que fica em um formato esquisito na volta.

Deixo o link do site no botão "Site" e o link para acessar o código que fiz está no botão "Códigos"
